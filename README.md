# XAI Machine Learning Models Repository

This repository contains implementations and explainability analysis (XAI) for multiple Machine Learning models.

## Repository Structure

| Notebook | Model |
|----------|-------|
| `XAI_PH2_EDA (2).ipynb` | Exploratory Data Analysis (EDA) |
| `XAI_DecisionTree_bassant_fixed (1).ipynb` | Decision Tree |
| `XAI_KNN_bassant_fixed (1).ipynb` | K-Nearest Neighbors (KNN) |
| `XAI_PH2_Logistic_reg(bassant) (1).ipynb` | Logistic Regression |
| `_ADA_XAI4_ (1).ipynb` | AdaBoost |
| `XAI_LightGBM_.ipynb` | LightGBM |
| `XAI_EDA_XGB_phase3_.ipynb` | XGBoost |
| `XAI_PH2_EDA_ET_Stack_NB (1).ipynb` | Extra Trees + Stacking + Naive Bayes |
